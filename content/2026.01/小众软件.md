# 小众软件

我乐意分享和总结我用的一些软件。[小众软件](https://www.appinn.com/) 其实是一个很有意思的分享站/论坛，网友们在这里分享各种软件，这群人（包括我在内）最大的共同点就是喜欢研究如何更高效有趣地使用电脑。

我把这些软件按照用户身份分类，比如区分普通用户和开发者，因为不同的身份会有不同的需求，就会需要不同的软件。

## 普通用户

### 桌面整理: Fences + Listary

Fences可以在桌面上创建分区，把桌面图标放到各个分区里面。对我来说这个应用最大的意义是，我是一个笔记本+显示器用户，只要拔插显示器桌面就会乱成一团糟，有了fences就不至于太乱，而且相比国内一些流氓软件的桌面分区工具，fences既没有广告也不会试图争夺我的注意力，大多数时候我都会忘记这个软件的存在。

往上说，“桌面分区”是“桌面美化”的手段之一，以前用 win10 的时候我还喜欢用全屏开始菜单+磁贴，win11虽然也有很多办法可以恢复全屏磁贴，但是 fences 我已经用习惯了。我还尝试过SAO Utils，相对比较小众，因为这是国人仿照刀剑神域动漫中出现的游戏菜单开发的工具，比较粉丝向和个性化。

启动器的选择有很多，everything，utools，还有powertoy里的那个小工具，最终我选择了listary，没有什么特别的原因，用习惯了，他们之间的差别并没有那么大。

找壁纸，目前来看我比较感兴趣的壁纸社区只有 Wallpaper Engine 或者 wallheaven，虽然我不喜欢用动态壁纸但是Wallpaper Engine毕竟还有一个那么活跃的社区。还有一些 linux 桌面用户的社区，比如 unixporn。

### 麦克风调整: SteelSeries GG

这本来是一个游戏软件，但是他有一个功能可以调整麦克风不同频率的响度，这样就可以很细节的调节自己麦克风输出的语音质量，而不只是被动接受一些语音软件的不知效果如何的降噪。


### 看书: Reeden

一个跨平台的阅读器，挺好用的，支持pdf，epub，txt，等格式。可以从telegram或者zlib下载要看的书，导入到reeden，用自己的网盘或者webdav或者S3进行同步，支持阅读统计，一些AI功能，以及很多DIY和美化。

值得一提的是Reeden作者的更新速度简直**世所罕见**。

### 魔法上网客户端: clash生态

桌面端我用sparkle，linux命令行用[clash-for-linux-install](https://github.com/nelvko/clash-for-linux-install)，移动端用 Clash Meta For Android。

clash的生态比较复杂，经常改名和跑路，目前最新的客户端就是 sparkle，确实好看也好用。

### 日记: Notion

这个其实蛮大众了我觉得，但还是分享一下。

我不用 Notion 的其他功能，也不记笔记，不拿来项目管理，也不拿来读书，只是写日记。我厌倦了对 all in one 的炒作，我怀疑那些自媒体到底用键盘亲自写下过多少字，工作、兴趣、生活最好还是分开。

## 开发

### 魔法上网: clash for linux install

[clash-for-linux-install](https://github.com/nelvko/clash-for-linux-install)，输入订阅链接就像桌面端一样开关系统代理、开关TUN。

### 内网穿透: cloudnet

自己搭建穿透又贵质量又一般，用cloudflare免费tunnel效果很差，sakurafrp速度和质量上限较高，但是下限也很低；cloudnet.world 便宜大碗质量高。

### 代码和项目管理: gitea + affine

gitea 不仅是代码仓库，也是各种软件包和容器的仓库，也有一定的 cicd 能力（部分兼容github actions），相比 gitlab 轻量许多。

我最近正在试用 affine，很适合我这种灵感迸发型选手进行项目管理，可以通过[这个视频](https://www.youtube.com/watch?v=ys627maBEiw)来了解affine。
