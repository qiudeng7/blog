## 框架已死论

[dead framework theory](https://aifoc.us/dead-framework-theory/), 框架已死论，认为目前所有的大模型都不得不尽力支持当下最流行的框架，如果你正在开发新的框架，你的竞争目标不是框架，而是大模型。这个信息来自阮一峰的科技爱好者周刊，但是我忘了是哪一期了，好在我找到了原文。

## 李新野

最近在folo的热门刷到“新的原野”这个博客，还是top1，他的博客地址： https://sinyalee.com/blog/ 。

点进去看了看，他让我想起了计算机圈，尤其是一些和算法竞赛相关的，甚至物理化学竞赛相关的圈子里的人，他们也常常和高学历和日系亚文化同时出现，尤其是性别认知方面的话题。

他们总是有一些比较极端的目标或者执念，在某一方面有相对出众的能力，往往有很强的优越感，但是又似乎承受着某种精神痛苦。

我不讨厌他们，我只是觉得这个群体很奇特，要是有针对这个话题的研究就好了，我会非常有兴趣了解。
